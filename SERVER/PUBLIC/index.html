Jigyasa Online Survey Management System API

Welcome to the Jigyasa API. This server provides endpoints for the Jigyasa Online Survey Management System.

Available Endpoints:

  • /api/auth - Authentication routes
  • /api/organizations - Organization management
  • /api/surveys - Survey management
  • /api/survey-responses - Survey responses
  • /api/data-analysis - Data analysis and visualizations
